# xale-c

**xale-c** is a really minimalistic text editor write in C.

## Getting started

Clone repo :

```
git clone https://github.com/axdelafuen/xale-c.git
```

Go to project folder :

```
cd xale-c/
```

Then compile the C program :

```
make
```

#### To uninstall

```
make clean
```
